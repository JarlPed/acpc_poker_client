Setup

To play the No-Limit bots using the Swordfish Gui, we will first have to accomplish two small setup steps.

Installation

The installation procedure is fairly straightforward. and can be achieved by these simple steps

  1. Step 1: Extract the tarball to a place on your machine. Ensure that the location you choose has read/write access.
  2. Step 2: Ensure that the following directories exist in the folder you uncrompessed the tarball to and that the file Swordfish.jar is in the Swordfish directory.
    • Swordfish
    • Swordfish/help
    • Swordfish/logs
    • Swordfish/libs
    • Swordfish/settings

The installation is now complete. To continue you must now configure your router to allow for port 4444 to be forwarded to this machine. If your computer is connected to the internet directly, or there are no other computers on your router, you may be able to move directly to the Playing the Bots section of this guide. If not continue on.

Configureing your Router

There is a great guide that can be located at http://portforward.com/routers.htm that explains how to do port forwarding for almost any router on the planet. It is used mostly to configure online games, but the steps are the same. We will be forwarding port 4444 to the machine that you are currently using the gui on. To find the ip that you need to forward this too, you can go to the network settings and check on the local machine. The ip will most likely be something like 192.168.X.Y. Once that is done we can now run the program.

Playing the Bots

Running the Gui

To run the gui, you can double click the Swordfish.jar file or open a terminal and type java -jar Swordfish.jar.
Once open, you should be prompted with a screen that ask you for some options. You can leave these the way they are or customize them for the game you wish to play. It is important however to choose the correct game of Short or Large Stack so the bot translation works.

Running the bot

If you are not running the bot code yourself, you can ignore most of this section, the only thing you will need is to obtain your Router's ip. This is different from your localmachine ip. You will have to tell this to the bot operator in order to play.

  1. Obtaining your IP

    To get this information you can visit http://www.ip-adress.com and your ip should appear on the page. You can also use any other method to get this information, you will just need it to run the bot.

  2. Starting the AAAI bot interface

    To start the bot, you will need a compiled version of the poker code. Then you will go to trunk/src/c and run the aaai_bot. The command to run the bot is ./aaai_bot NO_LIMIT 2 $IP 4444 strategy_player.so $STRATEGY where $IP is the ip address of the machine running the gui and $STRATEGY is a path to the strategy file you with so play. If you wish to play small stack, specify the -s flag at the end.

Troubleshooting

Other than any stack traces that you may get from running the bot in a terminal, there are a handful of logs in the Swordfish/logs folder. These can be deciphered or passed on to the bot handler to help diagnose your problem.

If you can't get the bot to connect toi the gui, chances are the port forwarding is incorrect or there has been a start-up error. Try following this guide again and if problems still presist contact the vendor.
